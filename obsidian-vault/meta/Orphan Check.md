# Orphan Check

How to spot weakly-connected notes in Graph View.

## How to use
1. Open **Graph View** (Ctrl+G).
2. Toggle **Show orphans** ON — any node with no links floats free; it needs links added.
3. Use the **Local Graph** (Ctrl+P → Open local graph) on a suspect note to see only its 1-hop neighbors.
4. Color by tag (gear → Groups) — clusters with a single thin bridge are weak points.

This vault was generated with **zero orphans** and every note ≤2 hops from [[00 - Project Map (MOC)]] — re-run the check after adding notes by hand.
